# Algoritmo Pilha e Fila

Pilha -> Princípio LIFO (Last In, First Out): O último elemento a entrar é o primeiro a sair.

Fila -> Princípio FIFO (First In, First Out): O primeiro elemento a entrar é o primeiro a sair.
